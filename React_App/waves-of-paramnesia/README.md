### NPM or YARN Commands

```
yarn start - To start the application

yarn build - To build the application

yarn test - To test the application

yarn lint - To check the lint errors in the application

yarn lint:fix - To check the lint errors and fix the code from the application

yarn format - To format the code in nicer way using prettier

yarn isReady - This is the combination of format and lint:fix

```
